Quotient — The Game

Resources, maps, and downloads for the interactive fiction adventure.

🎮 Interactive Fiction 📦 Release 108 🆔 IFID A8BCDC10-952A-48D7-A258-277BC72AEC72

Play an old-fashioned interactive fiction adventure set in the world of The Quantum Contingent and Quantum Time. Meet characters and visit locations from the books, solve puzzles, and carry out a secret mission — all in the tradition of classic text adventures like Zork.

Quotient was created with Inform 7. You can play it directly in your browser, or download the story file and run it in a desktop interpreter.

Desktop Interpreters

To play the downloaded story file on your computer, you'll need an interpreter. Any of these work well:

  • Gargoyle — macOS, Linux, or Windows
  • Spatterlight — macOS
  • Windows Frotz — Windows
  • Windows Glulxe — Windows
