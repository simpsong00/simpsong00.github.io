# Location Photos

Drop one photo per location here, named exactly by its **slug** with a `.jpg` extension
(e.g. `greenville-oh.jpg`). The globe references each photo at `/location-photos/{slug}.jpg`
and loads it lazily when a location card is opened.

**Tips**
- Aim for ~1200px on the long edge, JPEG quality ~80 ‚Äî keeps each file ~150‚Äì300KB.
- Same real place appears in both books a few times (see "reuse" notes) ‚Äî just copy the file to both names.
