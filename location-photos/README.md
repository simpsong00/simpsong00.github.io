# Location photo checklist

Drop each photo into this `location-photos/` folder named exactly `{id}.jpg` (the globe's
location slug). The globe loads each one lazily — only when its card is opened — and shows
a placeholder for any that are missing, so the globe always works even with photos absent.

**Tips**
- ~1200px on the long edge, JPEG quality ~80 → roughly 150–300KB each.
- A few real places appear in both books (noted _reuse_ below) — just copy the file to both names.
